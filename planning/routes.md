## Routes

- Browse -> GET -> /lists
- Read -> GET -> /lists/:id
- Edit -> POST -> /lists/:id/edit 
- Add -> POST -> /lists
- Delete -> POST -> /lists/:id/delete 

OR AJAX version:

- Browse -> GET -> /lists
- Read -> GET -> /lists/:id
- Edit -> PATCH -> /lists/:id
- Add -> POST -> /lists
- Delete -> DELETE -> /lists/:id
